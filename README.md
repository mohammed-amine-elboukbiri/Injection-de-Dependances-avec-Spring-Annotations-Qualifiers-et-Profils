# 🌱 Injection de Dépendances avec Spring : Annotations, Qualifiers et Profils

> Travaux Pratiques – Framework Spring | Maven | Java

---

## 📋 Description

Ce TP introduit les concepts fondamentaux de l'**injection de dépendances** avec le framework **Spring**. À travers des exercices progressifs, vous apprendrez à structurer une application Java en utilisant les mécanismes d'IoC (Inversion of Control) fournis par Spring.

---

## 🎯 Objectifs pédagogiques

À l'issue de ce TP, vous serez capable de :

- ✅ Créer un projet Spring avec **Maven**
- ✅ Définir des **interfaces** et leurs **implémentations**
- ✅ Utiliser les **annotations Spring** pour l'injection de dépendances (`@Component`, `@Autowired`, etc.)
- ✅ Configurer Spring pour scanner et détecter automatiquement les **beans** (`@ComponentScan`)
- ✅ Exécuter une application Spring avec injection de dépendances
- ✅ Spécifier quelle implémentation utiliser avec **`@Qualifier`**
- ✅ Utiliser les **profils Spring** (`@Profile`) pour activer différentes implémentations selon l'environnement
